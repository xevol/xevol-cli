# xevol

## Install

```bash
npm i -g xevol
```

## Usage

```bash
xevol login          # Authenticate via browser
xevol list           # List your transcriptions
xevol add <url>      # Transcribe a YouTube video
xevol view <id>      # View transcript
xevol spikes <id>    # View AI-generated insights
```
